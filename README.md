# Marina Dimitrova - Portfolio Website

A modern, responsive portfolio website for Marina Dimitrova, showcasing design work and creative projects.

## Project Overview

This is a static frontend portfolio website built with:
- **HTML5** - Semantic markup
- **CSS3** - Modern layouts with Grid and Flexbox
- **JavaScript (ES6)** - Minimal, modular interactivity

## Design System

**Colors:**
- Primary Red: `#8B2F2F`
- Cream Background: `#EDE5DD`
- Dark Text: `#2C2C2C`
- White: `#FFFFFF`

**Typography:**
- Headings: Playfair Display (serif)
- Body: Lora (serif)

## Project Structure

```
marina-portfolio/
├── index.html              # Main HTML file
├── css/
│   ├── base/              # Reset, variables, base styles
│   ├── layout/            # Grid systems, containers
│   ├── components/        # Buttons, cards, modals
│   └── sections/          # Hero, About, Gallery, Contact
├── js/
│   ├── modules/           # ES6 modules
│   └── main.js            # Entry point
├── assets/
│   ├── images/            # Project images
│   ├── backgrounds/       # Background graphics
│   ├── icons/             # Tool icons (Figma, Adobe, etc.)
│   └── profile/           # Profile photos
└── Materials/             # Original design files (not deployed)
```
